Projeto Manga & HQ

Uma plataforma completa para gerenciar coleções de mangás e HQs. Possui dashboard de progresso de leitura, integração com APIs externas (Jikan e Open Library) para busca automática de dados, sistema de notas e gestão de volumes possuídos, tudo com uma interface moderna e responsiva.
